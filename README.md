# Image Processor

Проект **Image Processor** представляет собой консольное приложение, которое позволяет применять к изображениям различные графические фильтры, аналогичные фильтрам в популярных графических редакторах.

## Описание

В этом проекте реализовано несколько фильтров для обработки изображений в формате **24-бит BMP**. Программа принимает изображение в формате BMP и применяет один или несколько фильтров, после чего сохраняет результат в указанный файл. 

Поддерживаемые фильтры:

- **Crop** — Обрезка изображения до заданных ширины и высоты.
- **Grayscale** — Преобразование изображения в оттенки серого.
- **Negative** — Преобразование изображения в негатив.
- **Sharpening** — Повышение резкости изображения.
- **Edge Detection** — Выделение границ изображения.
- **Gaussian Blur** — Размытие изображения с заданным значением сигмы.
- **Custom Filter** — Дополнительный фильтр, который можно выбрать и реализовать в проекте.

## Пример использования

### Формат аргументов командной строки:

```bash
{имя программы} {путь к входному файлу} {путь к выходному файлу} [-{имя фильтра 1} [параметр фильтра 1] [параметр фильтра 2] ...] [-{имя фильтра 2} [параметр фильтра 1] [параметр фильтра 2] ...] ...
```
### Пример запуска программы:
```bash
./image_processor input.bmp /tmp/output.bmp -crop 800 600 -gs -blur 0.5
```

## Список поддерживаемых фильтров:

Crop (-crop width height)
Обрезает изображение, начиная с верхнего левого угла. Если запрашиваемая ширина или высота превышают размеры исходного изображения, будет выведена доступная часть.

Grayscale (-gs)
Преобразует изображение в оттенки серого, используя стандартную формулу для яркости.

Negative (-neg)
Преобразует изображение в негатив, инвертируя значения пикселей.

Sharpening (-sharp)
Применяет фильтр повышения резкости изображения.

Edge Detection (-edge threshold)
Применяет фильтр для выделения границ. Изображение переводится в оттенки серого, и затем используется матрица для выделения границ.

Gaussian Blur (-blur sigma)
Применяет гауссово размытие с параметром сигмы.

Custom Filter (-custom <params>)
Дополнительный фильтр, создаваемый с учетом задания.

## Требования
Формат входных и выходных файлов: BMP (24-бит).

## Операционная система: Linux/Mac/Windows.

## Необходимые библиотеки: стандартная библиотека C++.

## Установка

Для сборки проекта необходимо иметь установленный CMake и компилятор C++ (например, g++).
Клонируйте репозиторий:

```bash
git clone https://github.com/poskrebish/image-processor.git
cd image-processor
```
Создайте директорию для сборки и соберите проект:
```bahs
mkdir build
cd build
cmake ..
make
```
Запустите программу:
```bash
./image_processor <input_file> <output_file> <filter_options>
```
